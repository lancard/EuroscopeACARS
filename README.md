# EuroscopeACARS (CPDLC)
Euroscope ACARS (CPDLC / TELEX) Receive and Send (sending is under development)

# How to use
1. Download latest release dll file
2. Load plugin in euroscope
3. Enter logon code (you can get logon code from hoppie's site)

# How to build and run
1. Run buildtool_install_for_local.bat
2. Run build.bat
3. It will generate EuroscopeACARS.dll
4. Load dll in Euroscope
